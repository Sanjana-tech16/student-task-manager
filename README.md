# Student Task Manager

A full-stack web application built with Flask and SQLite to help students manage their daily tasks.

## Features
* **User Authentication:** Secure Login and Registration.
* **Task Management:** Add, Update, and Delete tasks.
* **Data Persistence:** Uses SQLite database.
* **Responsive UI:** Styled with Bootstrap 5 and custom CSS.

## Installation
1.  Clone the repository.
2.  Create a virtual environment: `python -m venv venv`
3.  Install dependencies: `pip install -r requirements.txt`
4.  Initialize DB: `python setup_db.py`
5.  Run the app: `python app.py`

## Tech Stack
* **Backend:** Python, Flask
* **Database:** SQLite, SQLAlchemy
* **Frontend:** HTML5, CSS3, Bootstrap 5
